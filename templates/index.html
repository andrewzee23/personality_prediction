Natural Language Processing (NLP) is a branch of Artificial Intelligence (AI) that allows machines to understand and analyze human language. The Myers-Briggs Type Indicator ® is a tool that gives individuals a chance to determine their Personality Type. After cleaning a dataset of text and MBTI types obtained from Kaggle, running some machine learning models on the dataset, and plugging some code into Flask, we have created a simple method for determining a user’s personality type. Simply enter some text below, hit submit, and see what our machine learning model predicts for you.
